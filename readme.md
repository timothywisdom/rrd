-- Installation --

Prequisites:
1) Install NodeJS (version 6 or higher)
https://nodejs.org/en/

2) Install dotnet core (version 2 or higher)
https://www.microsoft.com/net/core#windowscmd

3) Install VS Code (recommended IDE)
https://code.visualstudio.com/
